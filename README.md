# ReadRoute Football Lab

An interactive football learning prototype for visualizing offensive concepts against common defensive coverages.

## Run it

Open `index.html` in a modern browser. No installation or server is required.

## Included

- Blank playbook with no prebuilt play concepts
- New unsaved plays always begin with empty routes and motions
- New Play and Save Play controls live inside the Create Play panel
- Named route options for each eligible receiver
- Route options can be assigned to multiple saved defenses
- Run Plays automatically selects matching route options for the chosen defense
- Route options branch from any clicked point on the base route
- Multiple dotted branches can be added to the same receiver
- All saved route options remain visible with direct edit and delete controls
- Formations can be mirrored into a new saved left/right version
- Plays can be mirrored into a chosen formation without changing its saved player positions
- Separate pre-snap motion paths that animate before post-snap assignments
- Click-to-draw custom routes for X, H, Y, Z, and RB
- Straight route segments with clickable sharp/rounded break points
- Per-segment route speed controls that apply only until the selected break point
- Live next-segment guide while drawing routes and movement
- Draggable route, motion, and defensive movement points
- Shift-lock for clean horizontal or vertical segments
- Visible route line immediately after the first route point
- Consistent route line thickness on short and long segments
- Per-route undo and clear controls
- Draw/Move board modes
- Drag repositioning for all 22 players
- Magnetic line-of-scrimmage snapping while moving players, with Alt to bypass
- Create Play and Run Plays workspaces
- Separate Create Formation, Create Play, and Create Defense screens
- Formation-folder playbook showing each saved play inside its formation
- Clean unsaved play drafts with no old-play selector
- Saved plays are edited only through Playbook Edit actions
- Playbook deletion controls for plays, empty formations, and defenses
- Reusable named offensive formations
- Named plays tied to their original formation
- Run-play choices filtered to plays manually created for the selected formation
- Custom defender movement paths
- Reactive zone defenders that hold leverage and respond to receivers entering their area
- Deep safeties carry the deepest threat when no defender provides help over the top
- Adjustable zone sizes for each defender with slider, typed value, and draggable top, bottom, left, and right handles
- In-editor previews for newly drawn offensive routes and defensive movement
- Defense editor hides offensive route lines
- Reusable named defenses with saved alignments and movement
- Editable labels for all 11 defensive players
- Blank 11-player defensive canvas with no coverage templates
- Draggable field notes with direct text editing and delete controls
- Auto-growing note boxes with compact top-right delete controls
- New formations and defenses inherit player labels from the first saved setup
- Smaller route and defensive movement arrowheads
- Independent play, formation, and defense selection in Run Plays
- Raised lime-ring selection indicator for active players
- Play/Reset animation for offense and defense
- Equal movement speed across every offensive and defensive path
- Dedicated Playbook tab with formation, play, and defense folders
- Full playbook export/import for moving saved work between browsers and computers
- Immediate autosaving after every edit
- Rotating recovery snapshots plus a secondary browser database backup
- Recovery of unfinished play drafts after refresh or accidental closure
- Manual snapshot restore and downloadable JSON backups
- Read-only diagrams when selecting files in the Playbook tab
- Run speeds at 100%, 75%, 50%, 25%, and 10%
- Temporary Run Plays scenario editing for player starts and movement paths
- Scenario changes never overwrite saved formations, plays, or defenses
- Defensive movement can be turned on or off for a Run Plays rep
- Saved defender-to-receiver man assignments
- Natural man pursuit with reaction delay, trailing leverage, momentum, and recovery speed
- Temporary man-assignment changes during Run Plays scenario editing
- Successful-save confirmation messages
- Notes hidden during Run Plays
- Editable display labels for each eligible receiver
- Full 11-player offense with quarterback and five offensive linemen
- Full 11-player defense with a four-man front in every coverage
- Proportioned NFL-width field, hashes, yard lines, and field numbers
- Selectable High School, College, and NFL hash/number standards
- Dynamic play and coverage diagrams
- Ranked teaching reads with coverage cues
- Browser-saved custom plays

The suggested reads are teaching prompts, not guarantees. Real rules vary by system, leverage, personnel, and post-snap rotation.
